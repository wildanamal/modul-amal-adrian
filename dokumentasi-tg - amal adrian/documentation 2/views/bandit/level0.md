## üß† Knowledge
Class selector digunakan untuk memilih elemen HTML berdasarkan
atribut `class`.

Class selector **selalu diawali dengan tanda titik (`.`)** diikuti
dengan nama class yang ingin dipilih.

Contoh penggunaan:
```css
.text {
  color: red;
}
